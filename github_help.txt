Command Line for Committing and Pushing to GitHub



After having git installed in the terminal 

- Go to the folder and  do 

git clone <link of repo>

- Create new branch to work on
git checkout -b <"name_branch">

- Add and commit a file
git add <file_name>

git commit <filename> -m <"Comment on what you have added">

- Push all changed to repo 
git push
